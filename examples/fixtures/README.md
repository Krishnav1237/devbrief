# DevBrief Fixtures

Small example projects for local testing and documentation.

Try them with:

```bash
npm run cli -- doctor --path examples/fixtures/npm-app
npm run cli -- doctor --path examples/fixtures/python-app
npm run cli -- doctor --path examples/fixtures/monorepo
npm run cli -- doctor --path examples/fixtures/docker-service
```

These fixtures are intentionally small. They exist to demonstrate project detection, output shape, and low-noise behavior.
